Stock Market Prediction Program

Natalia Navarrete

Overview

Purpose

This project aims to predict stock market trends using real time data.

Audience

The intended audience for this website includes investors, financial analysts, and anyone interested in stock market trends to make data-driven investments decisions.

Dynamic elements

The website will show dynamic elements like real-time stock data updates, charts and predictions created by C# backend with JavaScript and APIs to refresh the data and predictions live.

Content

Home page

This page will provide a quick summary overview of the stock market's top performing stocks and a search feauture to look up specific stocks predictions and historical data.

Stocks Details

This page will provide information about a specific stock. There will be a stock overview dsplaying current price, volume, industry/sector information, and 52-week high and low. There will so be a historical data table displaying historical results and a prediction summary with indicators.

Trends Visualization

This page will provide an interactive chart of the stock performace over time in the short and long run. Moreover, users can also compare trends of multiple stocks side by side.
